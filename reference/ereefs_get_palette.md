# Return Colour Palettes Associated With Specific Variables

Return Colour Palettes Associated With Specific Variables

## Usage

``` r
ereefs_get_palette(Variable)
```

## Arguments

- Variable:

  Character String. The Variable being mapped, currently accessible:
  "Turbidity", "Chlorophyll a", "DIN", "NH4", "NO3", "Secchi", "pH",
  "Wind", "True Colour"

## Value

A vector of hex code colour values of length 9

## Examples

``` r
if (FALSE)  #dont run because function is not exported
ereefs_get_palette("Turbidity")
 # \dontrun{}
```
