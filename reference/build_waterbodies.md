# Build the Marine Based Water Types Within the Northern Three Region

Build the Marine Based Water Types Within the Northern Three Region

## Usage

``` r
build_waterbodies(basins, named_islands)
```

## Arguments

- basins:

  An sf object produced by the
  [`build_basins()`](https://northern-3.github.io/RcTools/reference/build_basins.md)
  function

- named_islands:

  An sf object produced by the
  [`build_named_islands()`](https://northern-3.github.io/RcTools/reference/build_named_islands.md)
  function

## Value

An sf object

## Examples

``` r
if (FALSE)  #dont run because function is not exported
waterbodies <- build_waterbodies(basins, named_islands)
 # \dontrun{}
```
