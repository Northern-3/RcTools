# Return the Model's Min and Max Dates

Return the Model's Min and Max Dates

## Usage

``` r
ereefs_get_date_range(Model = "catalog")
```

## Arguments

- Model:

  Character string. Defaults to "catalog" however can also accept a
  premade url

## Value

Character string. Describes the min and max dates for the specific model

## Examples

``` r
if (FALSE)  #dont run because function requires user interaction
date_range <- ereefs_get_date_range()
 # \dontrun{}
```
