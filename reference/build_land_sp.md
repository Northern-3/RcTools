# Apply Special Transformations to the n3_land Dataset

Apply Special Transformations to the n3_land Dataset

## Usage

``` r
build_land_sp(n3_land)
```

## Arguments

- n3_land:

  An sf object produced within the
  [`build_n3_region()`](https://northern-3.github.io/RcTools/reference/build_n3_region.md)
  function

## Value

An sf object

## Examples

``` r
if (FALSE)  #dont run because function is not exported
n3_land <- build_land_sp(n3_land)
 # \dontrun{}
```
