# Provide a Range of Models to Choose From

Provide a Range of Models to Choose From

## Usage

``` r
ereefs_input_selection(input_file = "catalog")
```

## Arguments

- input_file:

  Character string. Defaults to "catalog" however can also accept a
  premade url

## Value

A Character string url to one of 12 models.

``` r
if (FALSE)  #dont run because function is not exported
```
