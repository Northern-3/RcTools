# Build the Marine Zone Boundaries Within the Northern Three Region

Build the Marine Zone Boundaries Within the Northern Three Region

## Usage

``` r
build_waterbody_boundaries(waterbodies)
```

## Arguments

- waterbodies:

  An sf object produced by the
  [`build_waterbodies()`](https://add-am.github.io/RcTools/reference/build_waterbodies.md)
  function

## Value

An sf object
