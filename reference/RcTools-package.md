# RcTools: A Collection of Report Card Tools

"This package makes a collection of Report Card functions available to
the technical team of the Northern Three Partnerships. Tools are focused
on the final steps of anaylsis and report writing, such as styling
tabular outputs for inclusion in the Technical Report, and providing
scoring functions and grading functions."
