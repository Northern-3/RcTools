# Build the Named Islands Within the Northern Three Region

Build the Named Islands Within the Northern Three Region

## Usage

``` r
build_named_islands(basins)
```

## Arguments

- basins:

  An sf object produced by the
  [`build_basins()`](https://northern-3.github.io/RcTools/reference/build_basins.md)
  function

## Value

An sf object

## Examples

``` r
if (FALSE)  #dont run because function is not exported
named_islands <- build_named_islands(basins)
 # \dontrun{}
```
